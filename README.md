# StayEase Sales - React Application

A React-based sales dashboard for StayEase to manage leads, resident allocation, and agreement generation.

## 🚀 Features

### 💼 Sales Dashboard
- **Lead Management**
  - View list of leads
  - Add new leads
  - Edit lead details
  - Delete leads
  - Track lead status

- **Resident Allocation**
  - View all residents
  - Allocate rooms to residents
  - Edit resident details
  - Delete residents
  - Track move-in/move-out

- **Agreement Generation**
  - Generate rental agreements
  - View agreement list
  - Edit agreement details
  - Delete agreements
  - Download agreements as PDF

## 🛠️ Technology Stack

- **Frontend:** React.js
- **State Management:** Context API
- **API Integration:** Axios
- **Styling:** Tailwind CSS
- **PDF Generation:** global html2pdf
- **Build Tool:** Create React App

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher) or yarn
- Git
